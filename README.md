## 用法

1. 输入Cookie [浏览器查看Cookie方法大全，网站Cookie查看方法，Chrome查看Cookie，IE查看Cookie_码农致富的博客-CSDN博客_ie 查看cookie](https://blog.csdn.net/u011781521/article/details/87791125)

   大致为打开网站，按F12，刷新网页，点击network/网络，找到/header/表头里的Cookie

2. 输入下载网址

3. 输入下载序列

4. 等待下载完成
